# О XEngl

##### Программа для помощи в заучивании слов английского языка, реализованная на языке C++  с помощью кроссплатформенной библиотеки элементов интерфейса GTK+.

# Сборка и запуск
```
$ git clone "https://github.com/luparev22/XEngl"
$ cd XEngl
$ make run
```
